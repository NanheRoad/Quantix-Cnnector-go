## 1. 介绍

MT-SICS（Mettler Toledo Standard Interface Command Set）是用于天平与计算机系统通信的标准命令集。

### 等级划分

| 等级 | 说明 |
|------|------|
| 0级 | 基础称重命令 |
| 1级 | 标准扩展命令 |
| 2级 | 设备相关命令 |
| 3级 | 应用扩展命令 |

特点：
- 0/1级命令在所有设备通用
- 推荐优先使用0/1级以保证兼容性

## 2. 数据交换基础

### 2.1 命令格式

```
命令 [参数] CRLF
```

规则：
- 使用大写字母
- 参数用空格分隔
- 必须以 CRLF 结尾

示例：
```
S
I1
D "HELLO"
```

---

### 2.2 应答格式

#### 带重量值

```
状态 重量 单位
```

示例：
```
S S 0.256 g
```

#### 无重量值

```
状态
```

---

#### 错误信息

| 代码 | 含义 |
|------|------|
| ES | 语法错误 |
| ET | 传输错误 |
| EL | 逻辑错误 |

## 3. 命令与应答

---

## 3.1 MT-SICS 0（基础命令）

### 查询类

| 命令 | 说明 |
|------|------|
| I0 | 查询所有支持的命令 |
| I1 | 查询协议等级和版本 |
| I2 | 查询设备信息 |
| I3 | 查询软件版本 |
| I4 | 查询序列号 |

---

### 称重类

#### S - 获取稳定重量

```
S
```

返回：
```
S S 100.00 g
```

---

#### SI - 立即获取重量

```
SI
```

特点：
- 不等待稳定
- 返回当前值

#### SIR - 连续输出重量

```
SIR
```

特点：
- 持续输出
- 可被 S / SI / @ 中断

---

### 控制类

#### Z - 归零

```
Z
```

#### ZI - 立即归零

```
ZI
```

#### @ - 复位

```
@
```

作用：
- 清除皮重
- 重置参数
- 恢复初始状态

---

## 3.2 MT-SICS 1（扩展命令）

### 显示控制

```
D "HELLO"
```

---

### 称皮相关

```
T   // 稳定称皮
TI  // 立即称皮
TAC // 清除皮重
TA  // 查询/设置皮重
```

---

### 键盘控制

```
K?1
K?2
K?3
K?4
```

---

### 自动发送

```
SR
```

说明：
- 重量变化时自动发送数据

---

## 3.3 MT-SICS 2（高级命令）

### 校准

```
C0 // 查询校准设置
C1 // 自动校准
C2 // 外部砝码校准
C3 // 内部砝码校准
```

---

### 时间与日期

```
DAT
TIM
```

---

### 设备信息

```
I10 // 设备标识
I11 // 型号
```

---

### 电源控制

```
PWR 0 // 待机
PWR 1 // 开机
```

---

### 参数设置

```
MW
```

可配置：
- 称重速度
- 稳定性
- 单位
- 自动归零
- 显示行为

---

### 数据输出扩展

```
SU   // 当前单位稳定值
SIU  // 当前单位立即值
SIRU // 当前单位连续输出
SRU  // 当前单位变化输出
```

---

## 3.4 MT-SICS 3（应用命令）

### 动态称重

```
SM1 // 单次动态称重
SM2 // 动态称重
SM3 // 动态连续称重
SM4 // 设置称重时间
```

---

## 开发建议

### 初始化

建议每次连接发送：

```
@
```

---

### 通信原则

- 必须解析返回值
- 所有命令都有响应
- 注意错误码处理

---

### 推荐使用范围

优先使用：

- MT-SICS 0
- MT-SICS 1

优点：

- 兼容性强
- 易于维护
- 适合跨设备系统
